The goal of omic.data is to run and visualize GO enrichments.

## Installation

You can install the development version of omic.data like so:

``` r
devtools::install_github("gurinina/omic.data")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(omic.data)
## basic example code
```
